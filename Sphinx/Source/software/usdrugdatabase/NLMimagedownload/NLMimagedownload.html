THE NLM Drug Image API

Downloading CSV Data from the National Library of Medicine

The application reads all the NDCs defined in the US drug database and downloads any visual information on the drugs.

The NLM has a drug image API available here:

http://rximage.nlm.nih.gov/docs/doku.php

A python application "getrximages.py" has been created to harvest this data.

This is then wrapped in a linux bash command "harvestrximages".

The following ENVIRONMENT variables are set in "workenv UVIS" and used by "getrximages.py" (actual paths will vary on different development computers).

  1. UVISUSDRUGIMAGEDIR=/media/kellerberrin/WorkDisk/Kellerberrin/Software/Active /Data/RawNDCDataFiles

    This environment variable identifies the data directory for "getrximages.py"

  2. UVISUSDRUGIMAGELOGFILE=/media/kellerberrin/WorkDisk/Kellerberrin/Software/Active /Data/RawNDCDataFiles/harvestrximages.log

    This environment variable identifies the log file of "harvestrximages.py"

  3. UVISUSDRUGDATAFILE=/media/kellerberrin/WorkDisk/Kellerberrin/Software/Active /Data/RawNDCDataFiles/NDCDatabase.csv

    This environment variable identifies the raw CSV file created using processrawNDCfiles.

    This file is read and the NDC codes in the file are then queried on the rximage API.

  4. UVISUSDRUGIMAGEMAXRATE=3

    A throttle that sets the maximum number of queries per second so the RxImage API hosted by NLM is not over-loaded.

  5. UVISUSDRUGIMAGEURL=

    The URL of the RxImage API hosted by the NLM.

Executing the CSV creation program.

  • harvertrximages Harvest the RxImage API from the National Library of Medicine.
